# Bedstack Simple

Simple version of [Bedstack](https://github.com/agnyz/bedstack) for rapid prototyping.

## Quickstart

1. Install dependencies

    ```bash
    bun install
    ```

2. Start the server

    ```bash
    bun dev
    ```
